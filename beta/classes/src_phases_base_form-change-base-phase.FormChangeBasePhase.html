    Class FormChangeBasePhaseAbstract

    A base phase to establish a "scene" for when a Pokemon changes form. Currently used for evolution and all toggleable form changes (e.g. Mega Evolution).

    Todo

    Use a separate Phaser Scene to contain assets. This currently injects sprites and other assets into FormChangeSceneUiHandler's container
