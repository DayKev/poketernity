    • getPokemonAtFieldIndex(): Pokemon | undefined

      Returns Pokemon | undefined

      The Pokemon at this Phase's field index, or undefined if no such Pokemon exists. Unlike getPokemon, this doesn't require the Pokemon to be active or on the field for it to be returned.

      Todo

      This is a bandaid fix for an issue where RecallPhase would crash the game during a double -> single battle transition. If this phase is refactored, this method should be removed in favor of a more flexible getPokemon
