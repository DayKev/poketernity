      Adds a new tag to the arena

      Parameters

      • newTagType: ArenaTagType

        The tag type being added

      • newTagSourceId: number

        The ID of the pokemon in play the tag came from (see BattleScene.getPokemonById)

      • newTagTurnCount: number = 0

        (Default 0) How many turns the tag lasts

      • OptionalnewTagSourceMove: MoveId

        (Optional) The move the tag came from, if applicable

      • newTagSide: ArenaTagSide = ArenaTagSide.BOTH

        (Default ArenaTagSide.BOTH) The side of the Arena the tag applies to

      • quiet: boolean = false

        (Default false) Whether a message should be queued on screen to announce the tag being added

      Returns boolean

      true if a new tag was added, or false if there already exists a tag of this type in the Arena

    • determineRerollIfLegendLike(
          pokemonSpecies: PokemonSpecies,
          level: number,
      ): boolean

      Determines whether or not to reroll for the given PokemonSpecies and level

      • Mega/Primal/Ultra legendaries and Arceus (720) cannot spawn below level 90
      • Most legendaries (including Regigigas but not Kyurem, Zacian and Zamazenta) cannot spawn below level 70
      • All other sublegends cannot spawn below level 50
      • The final base case only has Cosmoem/Cosmog

      Parameters
