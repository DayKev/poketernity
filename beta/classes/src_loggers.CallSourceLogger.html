    • new CallSourceLogger(ignoredFnNames: string[]): CallSourceLogger

      Constructs a logger that appends a line of the current execution's stack trace to the output.

      The line that gets outputted is the first line whose function is not listed in ignoredFnNames. (If no such line exists, the logger has no additional behaviour, and simply behaves as the regular console.)

      This logger can be useful for determining the source of a certain function call.

      Note: This extra functionality is currently only supported on Chrome and Firefox. On other browsers, it will most likely just behave as the regular console.

      Parameters

      • ignoredFnNames: string[]

        A list of function names to ignore.

      Returns CallSourceLogger
