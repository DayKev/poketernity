    setTurnOrder

    • setTurnOrder(order: FieldBattlerIndex[]): void

      Mocks the game's TurnCommandManager to set the specified turn order for future turns. Commands will execute in the given order regardless of command type, move priority, etc.

      Parameters

      • order: FieldBattlerIndex[]

        The turn order to set

      Returns void

      Example

      game.setTurnOrder([BattlerIndex.PLAYER, BattlerIndex.ENEMY, BattlerIndex.ENEMY_2, BattlerIndex.PLAYER_2]);

      Remarks

      To ensure stability, be sure to respect command type priority when mocking turn order. Not doing so can yield turn sequences that are not reproducible in normal gameplay and potentially lead to unexpected behavior.
